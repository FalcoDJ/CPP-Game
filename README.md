# CPP-Game

C++ Game Project

## Installation

### Linux

Linux Dependencies:

```bash
sudo apt-get install libsfml-dev cmake
```

On Linux, open the terminal type:

```bash
git clone https://github.com/FalcoDJ/CPP-Game.git
cd CPP-Game/
sudo ./install.sh
```
### Mac OS

Mac Dependencies:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
brew install cmake
brew install sfml
```
